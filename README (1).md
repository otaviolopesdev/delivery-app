# 🍔 QuickBite — App de Delivery de Comida

Sistema web de delivery de comida desenvolvido com HTML, CSS e JavaScript puro, simulando uma aplicação real de pedidos online.

## 🚀 Funcionalidades

- 🔐 Autenticação com login e cadastro de usuários
- 🗂️ Cardápio com filtro por categorias (Lanches, Pizzas, Japonesa, Bebidas e Sobremesas)
- 🛒 Carrinho lateral com controle de quantidade por item
- 💰 Cálculo automático do total do pedido
- ✅ Finalização de pedido com feedback visual
- 🔔 Notificações toast ao adicionar itens
- 💾 Persistência de usuários via localStorage

## 🛠️ Tecnologias utilizadas

- HTML5 (estrutura semântica)
- CSS3 (variáveis, animações, layout responsivo, grid)
- JavaScript ES6+ (DOM manipulation, localStorage, lógica de carrinho)

## 📸 Telas do sistema

| Tela | Descrição |
|------|-----------|
| Login | Autenticação com validação de campos |
| Cadastro | Criação de nova conta de usuário |
| Cardápio | Listagem de produtos com filtro por categoria |
| Carrinho | Gerenciamento dos itens selecionados |

## 📂 Como executar

1. Clone o repositório:
```bash
git clone https://github.com/otaviolopesdev/delivery-app.git
```

2. Abra o arquivo `index.html` no navegador

Nenhuma dependência externa necessária. Funciona 100% no browser.

## 💡 Aprendizados aplicados

- Componentização de interface com HTML/CSS puro
- Gerenciamento de estado com JavaScript vanilla
- Persistência de dados no client-side com localStorage
- UX com feedbacks visuais (toast, animações, estados de botão)
